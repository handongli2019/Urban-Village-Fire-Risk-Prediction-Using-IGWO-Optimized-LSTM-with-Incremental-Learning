# Results Folder

This folder stores exported prediction outputs generated by `main_GWO_LSTM_IL.m`.

Typical outputs include:

- `GWO_LSTM_predictions.xlsx`
- `GWO_LSTM_errors.xlsx`
- `GWO_LSTM_IL_predictions.xlsx`
- `GWO_LSTM_IL_errors.xlsx`

These files are not committed by default until the script is run.
